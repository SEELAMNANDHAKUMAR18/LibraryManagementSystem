# 📚 Library Management System (Java OOP)

## 📌 Objective

This project is developed as part of the Java Developer Internship Task 3.
It is a simple Library Management System that manages books and users using OOP concepts.

---

## 🛠 Technologies Used

* Java
* VS Code / IntelliJ
* Terminal

---

## 📂 Project Structure

* Book.java → Handles book details
* User.java → Handles user details
* Library.java → Core logic (issue/return books)
* Main.java → User interface (menu-driven)

---

## 🚀 Features

* Add books and users
* Display all books
* Issue a book
* Return a book

---

## 🧠 OOP Concepts Used

* Encapsulation → Private variables with getters
* Abstraction → Simplified methods (issueBook, returnBook)
* Polymorphism → Method usage
* Inheritance → (Can be extended further)

---

## ▶️ How to Run

1. Compile all files:

   ```
   javac *.java
   ```
2. Run:

   ```
   java Main
   ```
